Legacy Vault

Multi-chain estate planning for Bitcoin (BTC), Monero (XMR), and Stacks (STX). Secure your crypto for those who come next.

About the project

Legacy Vault is a full-stack web app for creating and managing estate plans: beneficiaries, allocation percentages, timelock policies, and optional wallet addresses. Built to be self-hosted and privacy-focused.

  • Rust API — Axum, SQLx, PostgreSQL, Argon2 auth, session cookies, rate limiting
  • Next.js frontend — React, TypeScript, Tailwind; responsive and accessible
  • Estate plans with optional BTC/XMR/STX addresses
  • Beneficiaries with allocation validation (sum ≤ 100%)
  • Timelock policies per plan
  • Admin: user list, audit log, role management

Tech stack

Rust Axum PostgreSQL Next.js TypeScript Tailwind CSS
