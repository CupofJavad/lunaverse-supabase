Predictive Modeling Equations

Transformer-based multi-modal prediction for financial, behavioral, and time-series data. BTC price predictor with CLI and web UI; runs locally or on server.

About the project

A Python implementation of core transformer/attention equations adapted for multi-modal predictive modeling. The app predicts Bitcoin (or similar datasets) using driver equations, volatility regime detection, event impacts, and uncertainty quantification. It includes a lightweight CLI menu and a web UI.

  • Core math: Scaled dot-product attention, multi-head attention, positional encoding, feed-forward layers
  • Multi-modal inputs: Numerical, temporal, behavioral; regression and classification
  • BTC price driver equation (reverse-engineered from historical drivers)
  • Volatility regime detection and uncertainty intervals (90%, 95%)
  • Local CLI admin menu and FastAPI web UI
  • Deployable on Lunaverse with app on external drive (Lacie) to save server disk

Tech stack

Python PyTorch FastAPI NumPy pandas scikit-learn
